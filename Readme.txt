Summary:

I created these demos for Visual Studio Live Orlando during December of 2011 (http://bit.ly/davedevvslive11).  The game is a great starter kit for creaintg your own XNA based Windows Phone game.  It utlizes a game loop, touch, accelerometer, sound and animations.
